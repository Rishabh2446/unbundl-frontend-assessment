# Unbundl Frontend Assessment

## Overview

This project is a React-based frontend application developed as part of the Unbundl Frontend Developer Assessment.

The application is built by converting the provided Figma design into a fully responsive React application using reusable components and clean CSS.

## Features

### Hero Section

* Responsive hero banner
* Lead generation form
* Offer strip
* Mobile-friendly layout

### Clinic CTA

* Call-to-action section for clinic booking
* Responsive design

### Get Your Perfect Smile in Four Simple Steps

* Data-driven rendering using reusable components
* Clean card layout

### Doctor-Led Section

* Informational content section
* Responsive image and text layout

### Whistle Difference

* Feature highlights
* Reusable data structure

### Dream Smiles Achieved Secretly

* Pricing and benefits section
* Responsive card design

### Results You'll Love

* Before and after smile transformation cards
* Dynamic rendering from external data file

### Footer CTA

* Booking action buttons
* Responsive layout

### Footer

* Navigation links
* Contact information
* Social media section

## Tech Stack

* React.js
* JavaScript (ES6+)
* CSS3
* Responsive Design

## Installation

Clone the repository:

```bash
git clone https://github.com/Rishabh2446/unbundl-frontend-assessment.git
```

Install dependencies:

```bash
npm install
```

Run development server:

```bash
npm start
```

## Production Build

```bash
npm run build
```
